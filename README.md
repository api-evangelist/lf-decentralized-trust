# LF Decentralized Trust (lf-decentralized-trust)
LF Decentralized Trust, formerly Hyperledger, is a Linux Foundation project umbrella for enterprise blockchain, distributed ledger technology, and decentralized identity solutions. It hosts frameworks like Hyperledger Fabric, Besu, and Aries for building secure, interoperable enterprise blockchain applications.
